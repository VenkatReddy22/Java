## ‚ú® Quick Reference

### **ARRAY BASICS**

```java
// Create
int[] arr = new int[5];              // Size 5, all 0s
int[] arr = {1, 2, 3, 4, 5};         // Size 5, values given

// Access
arr[0];                              // First element
arr.length;                          // Array size

// Modify
arr[0] = 10;                         // Update value
```

### **TIME COMPLEXITY CHEAT SHEET**

| **Operation**           | **Complexity** | **Notes**                 |
|--------------------------|----------------|---------------------------|
| Access by Index          | O(1)          | Instant via address math. |
| Search (Unsorted)        | O(n)          | Linear scan.              |
| Reverse                 | O(n)           | In-place requires O(1) space.    |
